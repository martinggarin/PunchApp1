This is an Expo React Native project. The following dependencies 
need to be installed:

expo: https://docs.expo.io/versions/latest/
    npm install --global expo-cli
Make sure to have the expo client app downloaded on your phone.

react navigation: https://reactnavigation.org/docs/getting-started

    npm install @react-navigation/native
    expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view
    npm install --save react-navigation-stack
    npm install react-navigation
    npm install --save react-native-screen

    npm install react-native-progress --save

Once all these dependencies have been installed you are ready to run the project.

In the project directory type: 
    expo start
or alternatively 
    npm start
